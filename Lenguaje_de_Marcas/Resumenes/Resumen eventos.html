1. ¿Cómo Escuchar Eventos? El Método Principal

addEventListener()

Es la forma moderna, recomendada y más potente de asignar un "escuchador" de eventos a un elemento.

Su sintaxis es: elemento.addEventListener('tipo_de_evento', funcion_a_ejecutar);

// 1. Seleccionamos el botón
const miBoton = document.getElementById('miBoton');

// 2. Definimos la función que se ejecutará al hacer click
function alHacerClick() {
    console.log('¡El botón fue presionado!');
    document.body.style.backgroundColor = '#f0f0f0';
}

// 3. Asignamos el escuchador de evento 'click'
miBoton.addEventListener('click', alHacerClick);
Nota: Existen formas antiguas como onclick en el HTML o elemento.onclick = ... en JS, pero addEventListener es superior porque permite añadir múltiples escuchadores al mismo evento sin sobrescribir los anteriores.

2. El Objeto `event`

Cuando un evento ocurre, la función que lo maneja recibe automáticamente un parámetro especial: el objeto event. Este objeto contiene toda la información sobre lo que acaba de suceder.

miBoton.addEventListener('click', function(event) {
    console.log(event); // Muestra un objeto con muchos detalles
    console.log(event.target); // Muestra el elemento exacto donde se originó el evento (el botón)
    console.log(event.type); // Muestra el tipo de evento ('click')
});

3. Controlando el Comportamiento de los Eventos

El objeto event nos da métodos para cambiar el flujo normal de un evento.

a) Detener la acción por defecto: `event.preventDefault()`

Evita que el navegador realice su acción predeterminada. Esencial para manejar formularios con JavaScript o prevenir que un enlace navegue.

const miFormulario = document.getElementById('miFormulario');

miFormulario.addEventListener('submit', function(event) {
    // Evita que el formulario se envíe y la página se recargue
    event.preventDefault(); 
    
    const input = document.querySelector('#miFormulario input');
    console.log('Formulario interceptado. Valor del campo:', input.value);
});

b) Detener la propagación: `event.stopPropagation()`

Los eventos "burbujean" (bubble up) desde el elemento más interno hacia sus padres. stopPropagation() detiene este burbujeo.

const divPadre = document.getElementById('padre');
const divHijo = document.getElementById('hijo');

divPadre.addEventListener('click', function() {
    console.log('Se ha hecho click en el DIV PADRE');
});

divHijo.addEventListener('click', function(event) {
    // Sin esta línea, el click en el hijo también activaría el click del padre.
    event.stopPropagation(); 
    
    console.log('Se ha hecho click en el DIV HIJO (y la propagación se ha detenido)');
});

4. Tipos de Eventos Comunes

Aquí tienes una lista de los eventos más utilizados, agrupados por categoría.

Eventos de Ratón (Mouse)

  • click: Un solo clic.
  • dblclick: Doble clic.
  • mousedown: Se presiona un botón del ratón sobre el elemento.
  • mouseup: Se suelta un botón del ratón sobre el elemento.
  • mousemove: El puntero se mueve sobre el elemento.
  • mouseover / mouseout: El puntero entra o sale del área de un elemento.

Eventos de Teclado (Keyboard)

  • keydown: Se presiona una tecla (se dispara repetidamente si se mantiene presionada).
  • keyup: Se suelta una tecla.
const campoTexto = document.getElementById('campoTexto');
campoTexto.addEventListener('keyup', function(event) {
    console.log(`Tecla soltada: ${event.key}, Código: ${event.code}`);
});
