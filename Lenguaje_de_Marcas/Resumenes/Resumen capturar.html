Guía Completa de Selección DOM con JavaScript

HTML de Ejemplo

Usaremos este código como base para todos los ejemplos.

1. Métodos Clásicos

a) getElementById()

Selecciona por el atributo id. Devuelve un solo elemento.

const titulo = document.getElementById('titulo-principal');

b) getElementsByTagName()

Selecciona por nombre de etiqueta (tag). Devuelve una HTMLCollection con todos los elementos que coinciden.

const todosLosParrafos = document.getElementsByTagName('p');

c) getElementsByClassName()

Selecciona por el atributo class. Devuelve una HTMLCollection.

const items = document.getElementsByClassName('item');

d) getElementsByName()

Selecciona por el atributo name, común en formularios. Devuelve una NodeList.

const campoUsuario = document.getElementsByName('usuario');

2. Métodos Modernos (¡Los más recomendados!)

a) querySelector()

Usa selectores de CSS para encontrar el primer elemento que coincida.

// Selección por ID
const tituloConQuery = document.querySelector('#titulo-principal');

// Selección compleja combinando clase y etiqueta
const itemImportante = document.querySelector('li.importante');

b) querySelectorAll()

Usa selectores de CSS para encontrar todos los elementos que coincidan. Devuelve una NodeList.

  •  dentro del elemento con id="lista-items"

    b) Hermanos: .nextElementSibling y .previousElementSibling

    Selecciona los elementos hermanos que están al mismo nivel.
