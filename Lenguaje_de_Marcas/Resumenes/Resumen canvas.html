Guía Completa de Canvas con JavaScript

1. Configuración Inicial del Canvas

Para empezar a dibujar, necesitas dos cosas: el elemento en tu HTML y obtener su "contexto" de renderizado 2D en JavaScript.

HTML Necesario:

JavaScript Inicial:

// 1. Seleccionar el canvas
const canvas = document.getElementById('miCanvas');

// 2. Obtener el contexto de dibujo 2D
const ctx = canvas.getContext('2d');

// ¡A partir de aquí, usamos 'ctx' para todas las operaciones de dibujo!

2. Dibujando Figuras Básicas

a) Rectángulos

Son las figuras más sencillas de dibujar.

// Estilo para el relleno
ctx.fillStyle = 'green';
// Dibuja un rectángulo relleno: (x, y, ancho, alto)
ctx.fillRect(25, 25, 100, 100);

// Estilo para el borde
ctx.strokeStyle = 'red';
// Dibuja el borde de un rectángulo: (x, y, ancho, alto)
ctx.strokeRect(150, 25, 100, 100);

b) Líneas y Trazados (Paths)

Para cualquier figura que no sea un rectángulo, usamos trazados. Es como dibujar con un lápiz: levantas, te mueves a un punto y empiezas a trazar.

ctx.beginPath(); // Inicia un nuevo trazado (levanta el lápiz)
ctx.moveTo(300, 50); // Mueve el lápiz a la posición inicial (x, y)
ctx.lineTo(400, 150); // Dibuja una línea hasta este punto
ctx.lineTo(300, 150); // Dibuja otra línea
ctx.closePath(); // Opcional: cierra el trazado volviendo al punto inicial
ctx.fillStyle = 'purple';
ctx.fill(); // Rellena la figura
ctx.stroke(); // Dibuja el borde

c) Círculos y Arcos

Se crean con el método arc().

ctx.beginPath();
// arc(x, y, radio, anguloInicio, anguloFinal)
// Los ángulos se miden en radianes. Math.PI * 2 es un círculo completo.
ctx.arc(500, 100, 50, 0, Math.PI * 2); 
ctx.fillStyle = 'orange';
ctx.fill();

3. Aplicando Estilos y Texto

Puedes controlar colores, grosores y fuentes.

// Grosor de la línea
ctx.lineWidth = 5;
ctx.strokeStyle = '#003366'; // Azul oscuro

// Dibuja un círculo solo con borde grueso
ctx.beginPath();
ctx.arc(75, 200, 40, 0, Math.PI * 2);
ctx.stroke();

// Escribir texto
ctx.font = '30px Arial';
ctx.fillStyle = 'black';
ctx.fillText('Hola Canvas!', 150, 215); // texto, x, y

4. Figuras Complejas (Combinando técnicas)

Aquí es donde la creatividad empieza. Combinamos figuras básicas para crear algo más reconocible.

a) Una Casa Sencilla

// Base de la casa (un rectángulo)
ctx.fillStyle = '#a52a2a'; // Marrón
ctx.fillRect(25, 270, 150, 110);

// Tejado (un trazado triangular)
ctx.beginPath();
ctx.moveTo(15, 270);
ctx.lineTo(100, 200);
ctx.lineTo(185, 270);
ctx.closePath();
ctx.fillStyle = '#6b4423'; // Marrón oscuro
ctx.fill();

b) Un Smiley

// Cara (círculo amarillo)
ctx.beginPath();
ctx.arc(300, 320, 50, 0, Math.PI * 2);
ctx.fillStyle = 'yellow';
ctx.fill();
ctx.stroke(); // Borde negro

// Ojos (rectángulos pequeños)
ctx.fillStyle = 'black';
ctx.fillRect(280, 300, 10, 15);
ctx.fillRect(310, 300, 10, 15);

// Sonrisa (un arco)
ctx.beginPath();
ctx.arc(300, 325, 35, 0, Math.PI); // Media vuelta
ctx.stroke();

c) Una Estrella (Uso de bucles y trigonometría)

Esto demuestra cómo se pueden crear figuras complejas de forma programática.

function dibujarEstrella(x, y, picos, radioExt, radioInt) {
    ctx.beginPath();
    ctx.moveTo(x, y - radioExt); // Mueve al pico superior
    for (let i = 0; i < picos; i++) {
        const anguloExt = Math.PI / picos * (2 * i + 1.5);
        const anguloInt = Math.PI / picos * (2 * i + 2.5);
        ctx.lineTo(x + Math.cos(anguloExt) * radioExt, y + Math.sin(anguloExt) * radioExt);
        ctx.lineTo(x + Math.cos(anguloInt) * radioInt, y + Math.sin(anguloInt) * radioInt);
    }
    ctx.closePath();
    ctx.fillStyle = 'gold';
    ctx.fill();
    ctx.stroke();
}

dibujarEstrella(450, 320, 5, 50, 25); // Dibuja una estrella de 5 picos

5. Dibujar Imágenes

También puedes cargar y dibujar imágenes en el canvas.

const miImagen = new Image();
miImagen.src = 'https://via.placeholder.com/100'; // URL de una imagen de ejemplo

// Importante: La imagen debe cargarse ANTES de dibujarla
miImagen.onload = function() {
    // drawImage(imagen, x, y, ancho, alto)
    ctx.drawImage(miImagen, 550, 270, 100, 100);
};
