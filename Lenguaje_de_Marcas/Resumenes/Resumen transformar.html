1. Ocultar y Mostrar Elementos

La forma más común de hacer desaparecer un elemento es cambiar su propiedad CSS display a 'none'. Para que vuelva a aparecer, se le devuelve su valor original (como 'block', 'flex', etc.).

const titulo = document.getElementById('tituloPrincipal');

// Función para alternar la visibilidad
function toggleVisibilidad() {
    if (titulo.style.display === 'none') {
        titulo.style.display = 'block'; // O el valor que tuviera por defecto
    } else {
        titulo.style.display = 'none';
    }
}

document.getElementById('toggleButton').addEventListener('click', toggleVisibilidad);

2. Crear y Añadir Nuevos Elementos

Crear un elemento desde cero es un proceso de 3 pasos:

  1. Crear el elemento: Con document.createElement('tag').
  2. Configurar el elemento: Añadirle texto, clases, atributos, etc.
  3. Añadirlo al DOM: Con parentElement.appendChild(newElement).
const lista = document.getElementById('listaElementos');
let contador = 2; // Para que cada elemento nuevo sea único

function anadirElemento() {
    // 1. Crear el elemento 
  •  en memoria
    const nuevoLi = document.createElement('li');
    
    // 2. Configurar el elemento
    nuevoLi.textContent = 'Elemento número ' + contador;
    
    // 3. Añadirlo como hijo del 
      
    lista.appendChild(nuevoLi);
    
    contador++; // Incrementamos para la próxima vez
}

document.getElementById('addElementButton').addEventListener('click', anadirElemento);

    • 3. Modificar Elementos Existentes

    a) Modificar Contenido de Texto

    La propiedad textContent es la forma más segura de cambiar solo el texto de un elemento, ya que ignora cualquier HTML.

    const parrafo = document.getElementById('parrafoPrincipal');

function cambiarTexto() {
    parrafo.textContent = '¡El texto ha sido modificado con JavaScript! La hora es: ' + new Date().toLocaleTimeString();
}

document.getElementById('changeTextButton').addEventListener('click', cambiarTexto);

    Alternativa: innerHTML permite insertar HTML, pero úsalo con cuidado, ya que puede ser un riesgo de seguridad si insertas contenido de un usuario sin sanearlo.

    b) Modificar Clases y Estilos

    El objeto classList es la mejor herramienta para gestionar clases de CSS. Permite añadir, quitar y alternar clases fácilmente.

    const parrafo = document.getElementById('parrafoPrincipal');

function cambiarEstilo() {
    // toggle: si la clase existe, la quita; si no, la añade.
    parrafo.classList.toggle('highlight');
}

document.getElementById('changeStyleButton').addEventListener('click', cambiarEstilo);

    4. Eliminar Elementos del DOM

    La forma moderna y sencilla de eliminar un elemento es llamar a su método .remove().

    const lista = document.getElementById('listaElementos');

function eliminarUltimoElemento() {
    // Seleccionamos el último elemento hijo de la lista
    const ultimoElemento = lista.lastElementChild;
    
    // Si existe un último elemento, lo eliminamos
    if (ultimoElemento) {
        ultimoElemento.remove();
    } else {
        alert('¡La lista ya está vacía!');
    }
}

document.getElementById('removeElementButton').addEventListener('click', eliminarUltimoElemento);
