Guía Rápida para Manejar XML con JavaScript

Nota: Aunque JSON es más común hoy en día, XML sigue siendo importante para muchas APIs (especialmente SOAP), archivos de configuración y sistemas heredados. El proceso para manejarlo es muy similar al de manejar HTML.

XML de Ejemplo

Imaginemos que tenemos este archivo libros.xml que queremos procesar.

El Proceso en 3 Pasos Clave

Para trabajar con XML en JavaScript, seguimos estos tres pasos:

Paso 1: Obtener el archivo XML

Usamos la API fetch para solicitar el archivo. Como el XML es texto, usamos el método .text() para obtener su contenido como una cadena (string).

// Usamos async/await para un código más limpio
async function obtenerXML() {
    const response = await fetch('ruta/a/tu/archivo.xml');
    const xmlTexto = await response.text();
    // Ahora 'xmlTexto' contiene el XML como un string gigante.
}

Paso 2: "Parsear" el XML a un Documento DOM

Un string no es útil para navegar. Necesitamos convertirlo en un objeto de tipo Documento, similar al DOM de HTML. Para esto, usamos DOMParser.

const parser = new DOMParser();
const xmlDoc = parser.parseFromString(xmlTexto, 'application/xml');
// ¡'xmlDoc' es ahora un documento con el que podemos trabajar!

Paso 3: Navegar y Extraer Datos (¡Igual que en HTML!)

Una vez parseado, puedes usar los mismos métodos que ya conoces de la guía de selección de elementos: querySelector, querySelectorAll, getElementsByTagName, etc.

// Seleccionar todos los elementos 
const libros = xmlDoc.querySelectorAll('libro');

// Recorrer cada libro para extraer su información
libros.forEach(libro => {
    // Obtener el atributo 'id'
    const id = libro.getAttribute('id');
    
    // Obtener el texto del elemento 
    const titulo = libro.querySelector('titulo').textContent;
    
    // Obtener el texto del elemento 
    const autor = libro.querySelector('autor').textContent;
    
    console.log(`ID: ${id}, Título: ${titulo}, Autor: ${autor}`);
});
